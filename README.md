# LiveKit-Info-Bot

An intelligent Voice AI agent that helps search employee email id and an answer general knowledge questions.

## Setup Instructions

### Prerequisites

- **Python 3.13** 
- **Node.js 18+** and npm
- API Keys:
  - Livekit API
  - Groq API
  - Google Gemini API

## Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/vishal27shetty/LiveKit-Info-Bot.git
   cd LiveKit-Info-Bot
   ```
